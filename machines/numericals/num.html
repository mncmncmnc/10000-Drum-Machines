This drum machine uses a variety of famous number sequences to generate patterns.


Instead of toggling individual steps, pick a sequence for each sound, try different combinations, and modify the overall sequence length to create interesting rhythms.


If you are a mathematician, and I got something wrong, please let me know! And if you are a mathematician (or not) and want to make a drum machine, we would love to have you as part of the 10,000 Drum Machines project.
