Typing / Phonetic Drum Machine

Type a drum sequence using these terms:
boom (bass drum) chit (snare) tick (closed hi-hat) tiss (open hi-hat) tcha (clap)
Use spaces to create rests.

Examples: boom   chit boom boomboom chit    boom tiss chit tiss 


×

This is a drum sequencer powered by typing out drum sounds as text and using spaces as a way to control rhythm.

It was inspired by Alex Mclean's live coding language Tidal Cycles, Allison Parrish's work with phonemes, conversations with Fran Rojo, and Sam Von Ehren's thoughts about typing games.

The sounds were designed and recorded from a Vermona DRM1 MKIII analog drum synthesizer.

It was made by Maxwell Neely-Cohen in 2025 and is the second entry in the 10,000 Drum Machines project.
